How to run this demos

1-. Install Node if you dont have it
2-. Press Window + r (It will open the "run" window)
3-. Type cmd and press enter (It will open command prompt)
4-. Navigate to the demos folder (you can navigate using the cd command example: cd c:/folder/sufolder/demos)
5-. Install npm modules with the following command
    npm i

6-. run the web server server.js file using the following command and press enter
    node server.js

7-. Now you can go to your browser and navigate to localhost/gameNameDemo
Example: localhost/railMatchDemo
